# Prompt Shortcuts

Use these in tmux panes.

## General

```text
Read AGENTS.md and workspace/PROJECT_CONTEXT.md. Follow all constraints. No commit.
```

```text
Next task. Follow AGENTS.md. No commit.
```

```text
Prepare a commit summary. Do not commit.
```

## PM

```text
Read AGENTS.md and workspace/TASK_BOARD.md. Break the next goal into small tasks and update the task board only.
```

## Designer

```text
Improve only the UI. Follow workspace/prompts/designer.md. Do not touch RAG files.
```

## Developer

```text
Implement the requested feature. Follow workspace/prompts/developer.md. Preserve RAG behavior and do not commit.
```

## QA

```text
QA current changes using workspace/REVIEW_CHECKLIST.md. Do not edit files.
```

## Docs

```text
Update docs only. Follow workspace/prompts/docs.md. Do not change app behavior.
```
